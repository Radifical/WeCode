Introduction to CSS

CSS stands Cascading Style Sheets. It is a computer coding language primarily used for structuring webpages made with HTML. In this article, I will go over the basics of styling with CSS.


The first thing you should do to begin adding CSS to your website is go to glitch.com and create an account. Once you are logged in, create a new hello website project and go to the style.css file. Now you can begin styling!


Let's start styling your webpage. At the top of your file, type body{}. In between the two curly brackets, write background-color: pink;. This has now changes the background of your whole webpage! Next you can write color: blue;, which changes the color of all text on your webpage to blue!


Next edit a specific element of the webpage by writing h1{color: red;}. Now all heading 1s will be red!


Finally, chenge the size of all images on your webpage by writing img{width: 150px; height: 200px;}. This has now changes the height and width of your image!


Congratulations! You have finished styling your webpage!
